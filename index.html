Introducción

Realizaremos un análisis inmunológico sobre muestras de sangre de once pacientes trasplantados renales que recibieron por 30 días un tratamiento inmunosupresor de mantenimiento de corticoides, micofenolato mofetil y belatacept. Este análisis se basa en definir las poblaciones de monocitos y linfocitos B. Para este se realizan inmunomarcaciones con anticuerpos monoclonales conjugados a los siguientes fluorocromos:

  • CD14 = PE Cy7
  • CD19 = Texas Red
  • APC = CD32, CD86.
  • APC-H7 = CD16, CD40
  • FITC = CD64, CD80

Por paciente tenemos tres tubos marcados de la siguiente manera:

1 CD14 CD19 ISOTIPO ISOTIPO ISOTIPO

Poblaciones de CD14+ y CD19+

Para cada tubo (tres por paciente) creamos “gates” para diferenciar las poblaciones CD14+ y CD19+. Las gates dividirán las poblaciones en cuatro ventanas para diferenciar las poblaciones + y - de CD14 y CD19.

gate1 = QuadGate((4.046e+03, 3.780e+03), ('PE-Cy7-A', 'PE-Texas Red-A'), region='bottom right', name='CD3+CD4-')
gate2 = QuadGate((4.046e+03, 3.780e+03), ('PE-Cy7-A', 'PE-Texas Red-A'), region='top right', name='CD3+CD4+')
gate3 = QuadGate((4.046e+03, 3.780e+03), ('PE-Cy7-A', 'PE-Texas Red-A'), region='top left', name='CD3-CD4+')
gate4 = QuadGate((4.046e+03, 3.780e+03), ('PE-Cy7-A', 'PE-Texas Red-A'), region='bottom left', name='CD3-CD4-')

Para identificar la población CD14+ realizamos la unión entre el Gate1 y Gate2. Para identificar la población CD19+ realizamos la unión entre la Gate2 y el Gate3. Para encontrar la población CD14+ y CD19+ realizamos la intersección entre las poblaciones anteriores, es decir buscamos el Gate2.

Dificultades al momento del análisis

Al momento del análisis encontré dificultades principalmente en elegir para las poblaciones menores (CD40, CD16, CD32, CD64, CD86 y CD80) el número que marca la división entre la población + y -. Solucioné esto al usar la función “values_count()”, una tabla que define la frecuencia de cada valor encontrado, en su posición [0] tiene el mayor valor, de esta manera pude encontrar el pico de la población.

Uso del control de isotipo

El control de isotipo es un control negativo que estima la unión no específica en el experimento, de esta manera estima la cantidad de “ruido” o error que tienen las mediciones de las poblaciones de menor cantidad (como por ejemplo CD40). Este control se consigue al aplicar un anticuerpo del misma especie y en la misma concentración que el utilizado para la marcación de la población analizada. En el caso de las poblaciones celulares CD14 y CD19 que utilizamos para distinguir las células B tenemos dos poblaciones celulares muy diferenciadas (negativa y positiva) para cada uno de estos marcadores, esto conlleva a que se distingan de forma efectiva las + y - y consecuentemente se pueda hacer un gating efectivo. Por estas razones, en este caso, no tenemos problema con la unión no específica y no necesitamos control de isotipo.
